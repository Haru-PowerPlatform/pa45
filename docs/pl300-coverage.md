1. 出題するたびに該当チェックボックスを `[x]` に更新
2. ミスが出たら `pl300-mistakes.md` ＋ `pl300/index.html` のカードに反映
3. 10問ごとに進捗サマリの「完了」列を更新してコミット
4. 領域バランスを見ながら出題（偏りを避ける）
5. 全チェック完了 = 試験準備完了の目安
