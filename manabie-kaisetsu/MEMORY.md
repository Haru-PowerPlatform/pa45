# マナビーテスト 解説サイト

## 概要

子（小6・男子）が受けるマナビーテスト（家庭学習研究社）の問題解説をブラウザで読める形にまとめた **学習支援サイト**。読むのは小6本人で、本人がサイトだけ見れば「解き直し」できる構成。

**Why:** 紙の解説冊子は分かりにくく、図がない問題で混乱する。タブレットで楽しく続けられる仕掛けが欲しいという親（さら）の要望。

**How to apply:** 新しいテストが返却されたら写真を見ながらこのサイトに新タブとして追加。形式は既存タブと統一。HTML編集はローカルとリポを両方更新してgit push。iPadは🔄ボタンで反映。

## 🎯 全タブ共通の問題表示フロー（必ず守る）

すべての問題は以下の表示フローに統一：

1. **初期表示**：問題文（h3 + q-text）のみ表示。下に「👀 答えを見る」ボタン
2. **「答えを見る」をタップ**：図／先生との会話／みちびき／正式な式／答え／チェックバーが一気に表示
3. **チェックバー**：「✅ 解けた」「🔁 直し問題にする」の2ボタン
4. **「🔁 直し問題にする」を押す**：localStorageに redo として保存 → 📋 ノートタブの目次にステータス表示

## 🎨 図のルール（必ず守る）

- 「のこりの○○」型の問題では **2本の線分図**
- **下の線分図は上の「残り」部分の幅にぴったり合わせる**（縦の点線で対応関係を表示）
- 1本目=「1」、2本目=「①」（マルイチ）で区別
- 「<b style='color:#dc2626'>残り</b>」は必ず **赤字** で強調
- 人名（A,B,C）は記号でなく「Aさんがもらった 2/5」のようにフルラベル
- フォント絵文字（⭕💨🔑等）は使用禁止（matplotlib豆腐化）→ ○×★●に置換

## 🗣 先生とさらの対話ルール

- 先生（青バッジ）とさら（オレンジバッジ）の会話形式で問題を導入
- **「逆算」など難しい用語を使わない**。苦手な子向けに段階的に
- 計算は **1行に詰めない**。`=` ごとに改行（`<br>`使用）
- 例：`1/3 + 2/5 = 5/15 + 6/15 = 11/15` ではなく：
  - 「3と5の最小公倍数は15だから… 1/3 = 5/15、2/5 = 6/15」
  - 「足してみて」
  - 「5/15 + 6/15 = 11/15」
- 先生は **なぜそれを計算するのか** を説明してから方法を聞く

## 図のフォント注意（豆腐対策）

matplotlibの日本語フォントは絵文字（⭕💨🔑🟡等）を描画できず**豆腐□**になる。`generate*.py` では以下に統一：

| 使ってはいけない | 使うべき |
|---|---|
| ⭕ | ○ |
| ✕ | × |
| 💨 🫧 | （削除して凡例で説明） |
| 🔑 ⭐ | ★ |
| 🟡🟢🔵🟠 | ●（背景色や文字色で区別） |

一括置換ツール: `C:\Users\isamu\Documents\rika6_kai5_figs\fix_tofu.py`（全generate*.pyを自動置換＆再実行）
