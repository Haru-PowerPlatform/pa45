📦 フローダウンロード

PA45ハンズオンで作成したPower Automateフローを配布しています。
ZIPをダウンロードして自分の環境にインポートするだけで使えます。

PA45 Solution Package
全フローまとめてダウンロード
第1〜6回のフロー6本を収録 · ソリューション形式（.zip）
⬇ ZIPをダウンロード
⬇️ インポート方法（ソリューション）
1
上のボタンからZIPをダウンロードしてPCに保存する
2
make.powerautomate.com を開き、左メニューの「ソリューション」をクリック
3
上部の「インポート」→「参照」でダウンロードしたZIPを選択→「次へ」
4
接続（Outlook・Teamsなど）を自分のアカウントに紐づけて「インポート」をクリック
5
インポート完了後、「ソリューション」→「PA45 Hands-on」の中に全フローが表示される
第1回
変数を初期化して使ってみよう
Initialize variable アクションで変数を作り、値をセット・メールで送信するシンプルなフロー。
手動トリガー Initialize variable Set variable メール送信
⬇ ZIPをダウンロード 解説ブログを読む →
第2回
変数を操作しよう（Set variable）
変数に値を代入・上書きし、条件によって内容を切り替える使い方を練習するフロー。
手動トリガー Initialize variable Set variable Compose
⬇ ZIPをダウンロード セッション詳細を見る →
第3回
条件分岐（Condition）で処理を分ける
Condition アクションで「もし〜なら」の分岐を作り、はい/いいえで異なるメールを送るフロー。
手動トリガー Initialize variable Condition メール送信
⬇ ZIPをダウンロード セッション詳細を見る →
第4回
Apply to each で繰り返し処理
SharePointリストのアイテムを1件ずつ取り出し、条件に合うものだけTeamsに通知するフロー。
手動トリガー Get items Apply to each Teams通知
⬇ ZIPをダウンロード セッション詳細を見る →
第5回
基礎固めWeek — 第1〜4回の総復習
変数・条件分岐・繰り返しを1本のフローに組み合わせた、基礎定着用の総合練習フロー。
手動トリガー Initialize variable Condition Apply to each
⬇ ZIPをダウンロード セッション詳細を見る →
第6回
Formsの回答が来たらお礼メールを送ろう
Microsoft Formsに回答が届いたら、回答者に自動でお礼メールを送信するフロー。
Forms 回答時 回答の詳細を取得 メール送信
